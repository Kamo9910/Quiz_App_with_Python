🧠 Quiz App (Python + Tkinter + API)

📌 Overview

This is a desktop Quiz Application built using Python, Object-Oriented Programming (OOP), and Tkinter for the graphical user interface.

The app dynamically fetches quiz questions from the Open Trivia Database API and presents them in an interactive format.

---

🚀 Features

- 🎯 Real-time quiz questions from an API
- 🖥️ GUI built with Tkinter
- 🧱 Structured using Object-Oriented Programming
- ✅ True/False question format
- 📊 Score tracking system
- 🔄 Automatic question loading

---

🛠️ Technologies Used

- Python
- Tkinter (GUI)
- Requests (API calls)
- OOP principles

---

📡 API Used

- Open Trivia Database
- Endpoint: https://opentdb.com/api.php

---

📂 Project Structure

quiz-app/
│── main.py
│── question_model.py
│── quiz_brain.py
│── ui.py
│── data.py

---

⚙️ How It Works

1. The app sends a request to the API to fetch quiz questions
2. Questions are converted into objects using OOP
3. The UI displays each question one at a time
4. User selects True/False
5. Score is updated based on answers

---

▶️ How to Run

1. Clone the repository:

git clone https://github.com/your-username/quiz-app.git

2. Navigate to the project folder:

cd quiz-app

3. Install dependencies:

pip install requests

4. Run the application:

python main.py
